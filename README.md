# HOT TAKES #

## Installation ##

Here are the dependancies you need to install:
- NodeJS
- Express
- Mongoose
- Multer


On Windows, these installations require to use PowerShell in administrator mode.


## Usage ##

Run `node server`. 

Use `Ctrl+C` in the terminal to stop the local server.
